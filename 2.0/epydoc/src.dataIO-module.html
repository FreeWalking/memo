Module dataIO

source code


该模块负责程序与文件数据的交互。
    1. 从文件中读取json
    2. 保存json进入文件

该模块基于的假设：
    1. 程序每次运行完毕时，把当日所有未完成和完成的任务写入date.json
        也就是会调用 write 函数

    testdata = {
        'datetime': '2014-12-28',
        'memolist': [
            {
                'id': '1',
                'title': 'homework',
                'starttime': '2012-12-28',
                'content': 'hello',
                'deadline': '2013-12-21',
                'finished': True
            }
        ],
    }

    configdata = {
        'lastdate': ''
    }
